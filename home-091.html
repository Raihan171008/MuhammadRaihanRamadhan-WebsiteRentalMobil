Deskripsi Mata Kuliah

Website ini dibuat sebagai soal UAS untuk mata kuliah Pemrograman Web. Mata kuliah ini membahas tentang:

  • Dasar-dasar pembuatan website dengan HTML, CSS, dan JavaScript
  • Konsep responsive design
  • Manipulasi DOM dengan JavaScript
  • Validasi form
  • Interaksi dengan pengguna

Proyek ini menerapkan semua konsep tersebut dalam bentuk website rental mobil yang terdiri dari beberapa halaman dengan fungsionalitas lengkap.
